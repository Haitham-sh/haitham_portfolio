Project Description

This application was developed as a full-stack project integrating frontend and backend components. It fetches live weather data from an external API and stores it alongside user entries in a local database. The backend was built with Node.js and Express framework to create custom API endpoints, while the frontend utilized vanilla JavaScript with HTML5 and CSS3 for a seamless user experience.

The project overcame two main challenges: establishing effective frontend-backend communication (solved using Fetch API and JSON), and presenting weather data in an attractive, readable format (achieved through an intuitive UI with visual weather indicators). The final product is a fully functional app delivering excellent performance and user experience.
